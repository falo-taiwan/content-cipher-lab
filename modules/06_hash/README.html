驗證與雜湊 (Hash)

【模組骨架】單向指紋與完整性校驗，學習 Hash $\neq$ Encryption 的不變真理。

⏳ 內容擴充中

本單元將規劃整理包含 MD5、SHA-1、SHA-256 與 SHA-512 的核心特性與單向特點，以及如何利用雜湊碼（Checksum）驗證下載檔案的原始狀態。

後續將由專案發起人指定詳細教材方向，並由開源社群共同開發實作展示工具。
