現代加密 (Encryption)

【模組骨架】實用檔案防護與高速對稱加密，捍衛現代資料流轉的安全邊界。

⏳ 內容擴充中

本單元將規劃整理包含 AES (AES-128/256) 的運作機制、為什麼它適合檔案傳輸與 ZIP/PDF 的加密保護，以及網路傳輸層安全協議 TLS/HTTPS 的基本原理。

後續將由專案發起人指定詳細教材方向，並由開源社群共同開發實作展示工具。
