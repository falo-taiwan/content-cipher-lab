古典密碼 (Classical Cipher)

【模組骨架】紙筆與幾何的交錯，歷史上的字元移位與替代科學。

⏳ 內容擴充中

本單元將規劃整理包含 Caesar Cipher、Vigenère Cipher、Atbash、Rail Fence、Polybius Square 與 Playfair Cipher 等古典加密方法的歷史故事與編解碼教學。

後續將由專案發起人指定詳細教材方向，並由開源社群共同開發實作展示工具。
