## 一、核心概念

**古典密碼（Classical Cipher）** 指的是歷史上使用紙筆即可完成編解碼的加解密方法。它們主要基於字元級別的兩個基本操作：
1. **替代（Substitution）**：將某個字母替換為另一個字母（例如凱撒密碼 Caesar Cipher）。
2. **移位（Transposition）**：重新排列字母的順序而不改變字母本身（例如柵欄密碼 Rail Fence Cipher）。

本單元屬於待擴充骨架，未來將由 **ff** 與團隊成員補充 Caesar Cipher、Vigenère Cipher、Atbash、Rail Fence、Polybius Square、與 Playfair Cipher 的互動演算法及 POC 教學。
