# A. 內容隱藏 (Steganography)

## 一、核心概念

**內容隱藏（Steganography，又稱隱寫術）** 的核心目的不是讓訊息變得「看不懂」，而是**隱藏「訊息存在」的這件事實**。 

這與傳統加密（Encryption）有根本性的不同：
* **加密（Encryption）**：旁人知道你在傳遞秘密，但不知道秘密的內容是什麼（產生亂碼）。
* **隱寫術（Steganography）**：旁人看來是一幅普通的畫、一首普通的詩或一段無意義的長篇文章，根本不知道裡面藏有秘密。

在 AI 時代，隱寫術被廣泛討論於浮水印（AI Watermarking）、Prompt 隱藏指令（Hidden Instructions）以及 Agent 之間的安全隱密通道建構。

⸻

## 二、研究與展示範疇

本單元主要整理與展示以下幾種內容隱藏技術：

1. **藏頭詩 / 藏尾詩**：利用中文文學結構，在詩句的固定位置（通常是首字或尾字）藏入真實的訊息。
2. **聖經密碼 (Bible Code / ELS)**：等距取字（Equidistant Letter Sequence）。從一段看似普通的長篇經文或文章中，以固定的間隔步長（如每隔 5 個字）抽取字元，拼湊出隱藏的訊息。
3. **固定間隔取字**：在任意文章中，使用預先約定的數學規律（例如費波那契數列間隔、質數間隔）進行字元抽取。
4. **媒體隱寫術 (Media Steganography)**：
   - **圖片藏字 / LSB 隱寫**：修改影像像素的最低有效位（Least Significant Bit），在肉眼無法察覺的色彩微調中塞入大量文字。
   - **EXIF Metadata**：將密文或指令隱藏在圖片的拍攝參數（如相機型號、GPS 座標、時間戳記）中。
   - **Audio / Video 隱寫**：在音訊或影片的雜訊軌道中嵌入簽名。
5. **QR Hybrid (混合 QR 碼)**：在標準 QR 碼中，透過微調網格點或對比度，塞入第二層只有特定掃描器才能解開的保密資訊。

⸻

## 三、教學重點與思考

1. **安全性高低**：隱寫術的安全性完全依賴於「演算法的保密性」或「秘密間隔的保密性」。一旦旁人知道了取字規律，隱寫術就完全失效。
2. **AI 的理解能力**：大語言模型（LLM）非常擅長處理「等距取字」與「藏頭詩」的還原嗎？實際上，現有的 LLM 對於精確的字元計數與字元偏移（Character Offset）常有幻覺，這使得隱寫術在防範 AI 爬蟲或對抗 AI 稽核時，成為一種有趣的探索方向。
