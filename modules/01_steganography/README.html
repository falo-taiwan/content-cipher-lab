內容隱藏 (Steganography)

「看不見，才是最高明的保密方式。」在尋常中藏匿真相的隱寫藝術。

01

什麼是隱寫術？

隱寫術（Steganography）與傳統密碼學（Cryptography）不同。密碼學專注於讓訊息變得無法解讀（變成亂碼），而隱寫術則專注於讓訊息的存在本身不被察覺。

不論是古希臘人在奴隸剃光頭後刻字再等頭髮長出，還是中文古典詩詞中的「藏頭詩」，都在利用無害的載體包裹著關鍵秘密。在 AI Workflow 的安全控制中，隱寫術常被用於防範資料外洩的浮水印技術。

02

聖經密碼 (Bible Code / ELS)

等距取字（Equidistant Letter Sequence, 簡稱 ELS）是隱寫術的一種經典形式。透過跳過固定數量的字元，從看似隨機或具有宗教意義的文本中萃取出密碼。

此項研究在無神論或宗教真偽之外，提供了一種極佳的資訊嵌字示範：只要文本容量夠大，總能藉由調整間隔長度（步長 $N$）拼湊出特定的字詞。這有助於學員理解 AI 時代下如何從語音、語意噪訊中提煉特定指令。
