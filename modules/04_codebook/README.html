密碼書系統 (Codebook Cipher)

「數字代替語言，對照表即是天機。」回顧情報史上的語意替換技術。

01

語意層面的編碼代換

與 Caesar 或 Vigenère 等對單個字元進行位移的演算法不同，密碼書（Codebook） 是以「詞彙」或「完整語意」為單元進行代換的。這使它在防範經典密碼分析（如字母頻率統計）時，擁有高得多的安全性。

然而，密碼書的致命缺陷在於物理分發：一旦對照本遺失、被偷拍或被策反，所有安全防護立刻土崩瓦解。

02

歷史通訊的影子

在國共戰爭、外交電報及二戰的商船通訊中，密碼書被極為廣泛地運用。透過編製包含數千個常用中文詞彙的數字代號本，通訊兵能夠將明文電報縮編為一串串數字密碼發送。

這項技術在 AI 時代的映射，正是大型語言模型中的 Tokenizer 機制：將自然語言切分為數以萬計的 Token ID，再傳輸給模型運算。
